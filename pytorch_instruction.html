1. 🗺️ 核心概念速览

1.1 PyTorch 是什么

PyTorch 是当前最流行的深度学习框架之一，由 Meta AI 开发维护。核心特点：
• 动态计算图（Define-by-Run）— 每次前向传播都可以构建不同的图，调试友好
• Pythonic — 与 NumPy 高度相似，无缝融入 Python 生态
• 强大的 GPU 加速 — 一行代码在 CPU/GPU 间切换
• 丰富的生态系统 — torchvision, torchaudio, torchtext, HuggingFace, PyTorch Lightning

1.3 动态计算图 vs 静态计算图

特性PyTorch（动态图）TensorFlow 1.x（静态图）
构建方式每步即时执行，随写随运行（Eager Mode）先定义完整图，再统一执行
调试✅ 可用 Python 调试器、print 任意位置❌ 难以调试，图构建和执行分离
条件/循环✅ 原生 Python if/for，灵活需要特殊控制流 API（tf.cond, tf.while）
性能较优（torch.compile 正在追赶）优化空间更大
部署TorchScript / ONNX / torch.exportSavedModel / TFLite

1.4 最小训练代码骨架

import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader, TensorDataset

# 1. 数据
X = torch.randn(1000, 10)
y = torch.randint(0, 2, (1000,))
loader = DataLoader(TensorDataset(X, y), batch_size=32, shuffle=True)

# 2. 模型
model = nn.Sequential(nn.Linear(10, 64), nn.ReLU(), nn.Linear(64, 2))

# 3. 损失 + 优化器
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=1e-3)

# 4. 训练循环
for epoch in range(10):
    for xb, yb in loader:
        pred = model(xb)               # 前向传播
        loss = criterion(pred, yb)      # 计算损失
        optimizer.zero_grad()           # 梯度清零
        loss.backward()                 # 反向传播
        optimizer.step()                # 更新参数
    print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")

3. 📐 自动求导 (Autograd)

PyTorch 的自动求导引擎是所有神经网络训练的基石。它在后台构建计算图，backward() 时自动计算所有参数的梯度。
🎯 梯度追踪与计算 核心
🎬 场景⌨️ 代码📝 说明
开启梯度追踪x = torch.tensor([2.0], requires_grad=True)创建时指定
稍后开启x.requires_grad_(True)下划线后缀 = in-place 修改
反向传播loss.backward()标量调用，计算所有 requires_grad=True 的梯度
获取梯度x.grad计算后的梯度值
梯度清零optimizer.zero_grad()⭐ 每个训练 step 前必须清零
手动清零x.grad.zero_()手动清零某个张量的梯度
禁止梯度追踪with torch.no_grad():⭐ 推理/评估时必须用，省显存+加速
分离计算图x.detach()返回一个新张量，与计算图断开
梯度裁剪torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)防止梯度爆炸（RNN 常用）
计算梯度范数torch.nn.utils.clip_grad_norm_ 不裁剪只计算监控梯度是否正常
🎯 关键概念详解 进阶

requires_grad 传播规则

a = torch.randn(3, requires_grad=True)
b = a * 2           # b.requires_grad → True（继承）
c = b.detach()      # c.requires_grad → False（断开）
d = c + 1           # d.requires_grad → False

no_grad vs eval()

操作作用何时用
torch.no_grad()禁用 autograd 图构建推理、验证、评估（省显存）
model.eval()设置模型为评估模式影响 Dropout（关闭）、BatchNorm（用全局统计）
model.train()设置模型为训练模式恢复正常训练行为
⚠️
常见陷阱：验证/推理时两个都要用 — model.eval() 切换行为，torch.no_grad() 关闭梯度记录。只用一个是不够的！

4. 📦 数据加载

🎯 Dataset & DataLoader 核心

💡 一句话：Dataset 定义"怎么读取一个样本"，DataLoader 负责"如何批量、多线程地喂给模型"。

内置 TensorDataset

from torch.utils.data import DataLoader, TensorDataset

X = torch.randn(1000, 28, 28)     # 1000 个样本
y = torch.randint(0, 10, (1000,))

dataset = TensorDataset(X, y)
loader = DataLoader(dataset, batch_size=32, shuffle=True, num_workers=4)

自定义 Dataset（模板）

from torch.utils.data import Dataset

class MyDataset(Dataset):
    def __init__(self, data_dir, transform=None):
        self.data = ...                         # 加载文件列表或数据
        self.transform = transform

    def __len__(self):

    def __getitem__(self, idx):
        x = self.data[idx]                      # 读取第 idx 个样本

        return x, label

DataLoader 关键参数

参数默认说明
batch_size1每批样本数

5. 🧠 网络层 nn.Module

🎯 nn.Module 模板 核心

💡 一句话：所有神经网络的基类。__init__ 中定义层，forward 中定义前向逻辑。

import torch.nn as nn
import torch.nn.functional as F

class MyNet(nn.Module):
    def __init__(self, in_dim, hidden_dim, num_classes):
        super().__init__()
        self.fc1 = nn.Linear(in_dim, hidden_dim)
        self.bn = nn.BatchNorm1d(hidden_dim)
        self.dropout = nn.Dropout(0.3)
        self.fc2 = nn.Linear(hidden_dim, num_classes)

    def forward(self, x):
        x = self.fc1(x)
        x = self.bn(x)
        x = F.relu(x)
        x = self.dropout(x)
        x = self.fc2(x)
        return x

model = MyNet(784, 256, 10)
print(model)  # 打印完整结构
print(f"参数量: {sum(p.numel() for p in model.parameters()):,}")
🎯 常用层速查 核心
分类层核心参数
全连接nn.Linear(in, out)输入/输出维度
nn.Identity()占位层（不做任何转换）
卷积nn.Conv2d(in_c, out_c, k, s, p)输入/输出通道、卷积核尺寸、步长、填充
nn.ConvTranspose2d(...)转置卷积（上采样）
nn.MaxPool2d(k) / nn.AdaptiveAvgPool2d((1,1))池化（后者自动适配到目标尺寸）
RNNnn.LSTM(in, hidden, layers)输入维度、隐藏维度、层数
nn.GRU(in, hidden, layers)参数更少，训练更快
Transformernn.TransformerEncoderLayer(d_model, nhead)模型维度、注意力头数
归一化nn.BatchNorm1d/2d(num_features)批归一化
nn.LayerNorm(normalized_shape)层归一化（Transformer 标配）
nn.GroupNorm(num_groups, num_channels)组归一化（小 batch_size 时替代 BN）
正则化nn.Dropout(p=0.5)随机失活
nn.Dropout2d(p=0.5)整通道随机失活（卷积用）
激活nn.ReLU() / F.relu()ReLU，注意 inplace=True 可省内存
nn.GELU() / nn.SiLU()Transformer 常用激活
